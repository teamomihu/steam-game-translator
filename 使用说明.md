# Steam 游戏汉化工具 - 使用说明

## 快速开始（3步）

### 第1步：启动程序

**macOS 用户：**

打开「终端」应用（在启动台搜索"终端"），然后逐行复制粘贴以下命令：

```
cd ~/work/ai-know/translate
./start.sh
```

> 首次运行会自动安装所需的环境，大约需要 2-5 分钟，请耐心等待。
> 之后每次启动只需几秒钟。

---

### 第2步：配置翻译引擎

程序启动后，你需要选择一个翻译引擎。有两种方案：

#### 方案A：使用 Ollama（推荐，完全免费，不需要联网）

1. 先安装 Ollama：访问 https://ollama.com/download 下载安装
2. 安装后打开终端，运行：
   ```
   ollama pull qwen2.5:7b
   ```
   等它下载完成（大约 4GB，需要几分钟）
3. 在程序界面的「设置」区域，把翻译引擎改为 **ollama**
4. 完成！可以开始翻译了

#### 方案B：使用 OpenAI（效果最好，需要付费 API Key）

1. 访问 https://platform.openai.com/api-keys 注册并获取 API Key
2. 启动前在终端运行：
   ```
   export OPENAI_API_KEY="sk-你的key粘贴到这里"
   ```
3. 然后再运行 `./start.sh` 启动程序
4. 翻译引擎保持默认的 **openai** 即可

#### 方案C：使用 DeepL（速度快，免费额度50万字/月）

1. 访问 https://www.deepl.com/pro-api 注册免费账号，获取 API Key
2. 启动前在终端运行：
   ```
   export DEEPL_API_KEY="你的key粘贴到这里"
   ```
3. 在程序中翻译引擎改为 **deepl**

---

### 第3步：开始翻译

1. 先打开你要翻译的 Steam 游戏（建议用**窗口模式**运行游戏）
2. 在汉化工具中点击 **「框选区域」** 按钮
3. 屏幕会变暗，用鼠标**拖出一个框**，框住游戏中你要翻译的文字区域
4. 松开鼠标，区域就选好了
5. 然后：
   - 点 **「截图翻译」**：翻译一次当前画面
   - 点 **「开始实时翻译」**：自动持续翻译（游戏画面变化时会自动重新翻译）

翻译结果会显示在：
- 程序底部的「翻译结果」区域（文字版）
- 游戏画面上方的透明覆盖层（直接覆盖在原文上）

## 常见问题

### Q: 启动报错 "没有找到 Python"
先安装 Python。打开终端运行：
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install python@3.13
```

### Q: 翻译结果是空的 / "未识别到文字"
- 确保框选的区域里确实有文字
- 试试把「OCR放大倍数」调到 3 或 4（小字效果更好）
- 确保游戏用**窗口模式**运行（全屏独占模式可能截不到图）

### Q: 翻译报错 / 翻译失败
- 检查翻译引擎的 API Key 是否正确设置
- 如果用 Ollama，确保 Ollama 正在运行（终端运行 `ollama serve`）
- 检查网络连接（OpenAI/DeepL 需要联网）

### Q: 怎么关闭程序？
- 关闭主窗口后程序会最小化到系统托盘（屏幕右上角的图标区域）
- 右键点击托盘图标，选择「退出」即可完全关闭

### Q: 配置文件在哪里？
在 `~/.steam-translator/config.json`，你可以用文本编辑器打开修改。
